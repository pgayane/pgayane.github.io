2013 - McGill era: Discovering Relevant Tutorial section

Tech: Java, Stanford CoreNLP, R
This project is based on my master thesis research at STENO group, McGill University. This work focuses on discovering relevant sections of tutorials for a given API type. We approach this problem by identifying API types in API tutorial, dividing the tutorial into small fragments and classifying them based on linguistic and structural features. The system we developed can ease information discovery for the developers who need information about a particular API type.

The source code can be found at my GitHub account. Hopefully I will refactor the source code and add more documentation in near future so that it is usable without me.

2012 - McGill era: Mining Answers to Technical Forum Questions

Tech: Java, OpenNLP, Lucene
This was a project done for Recommender Systems for Software Engineering course which led to my master thesis research. The project was able to filter "How to" questions and select the most important words and phrases out of the sentence and afterwards using Lucene search the index of javadocs to find the potential answer. During this project I learnt a lot about using NLP tools and the challenges specific for Software Documentation which help me in my future research.

To read more about this research feel free to to read the final report

2012 - McGill era: Smart Video Player

Tech: Java, JMF, JavaCV
Smart Video Player was created as a team project for Computer Vision course at McGill University. The video player is able to track multiple faces in real-time, distinguish the poses of the faces and perform start/pause/stop actions based on pose of the face. For example if person turns head then the video would pause and resume if face turns back. If person leaves the seat then video is stopped.

More details, the demo and the source code can be found here.

2011 - McGill era: Detecting teams from individual player trajectories

Tech: MatLab, Java
This is a course project for Machine Learning and my first project at McGill University. The motivation of the project is that in many multiplayer online role-playing games when players engage in secondary communication or form hidden aliases is considered cheating. This project uses unsupervised machine learning techniques to distinguish actual teams players by looking into the actions the players took. It uses logs from large scale multiplayer game research framework developed at McGill University (Mammonth).

For more info you can read the report or the presentation.
